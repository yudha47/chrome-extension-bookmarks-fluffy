Bookmarks Fluffy

Privacy Policy

Last updated: June 7, 2026  ·  Version 1.0.0  ·  Effective immediately upon installation

Short version: Bookmarks Fluffy does not collect, transmit, or share any of your data. Everything stays in your browser — always.

1. Overview

Bookmarks Fluffy ("the Extension") is a Chrome Extension built to provide a modern interface for managing Chrome bookmarks. This Privacy Policy explains what data the Extension accesses, how it is used, and what it does not do.

By installing and using Bookmarks Fluffy, you agree to the terms described in this policy.

2. Data the Extension Accesses

Data Type Why Transmitted?
Chrome Bookmarks To display, search, and navigate your bookmarks Never
Pinned bookmark IDs To remember which bookmarks you pinned Never
Recently opened bookmarks To show your recent bookmark history Never
Theme preference To remember dark or light mode setting Never
Active tab URL To open the Side Panel in the current tab context Never

3. Data Storage

The Extension stores a small amount of data locally on your device using chrome.storage.local. This storage is:

  • Stored only on your local machine
  • Never synced to external servers
  • Never shared with third parties
  • Removed completely when you uninstall the Extension

Data stored locally:

  • pinned — array of bookmark IDs you manually pinned
  • recent — list of up to 20 recently opened bookmarks (ID, title, URL, timestamp)
  • theme — your preferred color theme ("dark" or "light")

4. What We Do NOT Do

  • No   Collect personal information
  • No   Transmit data to any server
  • No   Use analytics or tracking tools
  • No   Sell or share data with third parties
  • No   Access browsing history beyond what you open via the Extension
  • No   Read page content of any website
  • No   Use cookies

5. Third-Party Services

The Extension fetches website favicons (small site icons) using the Google Favicon API (https://www.google.com/s2/favicons). This is a public Google service. When a favicon is loaded, Google's servers may receive the domain name of the bookmark. No personal data is included in this request.

This is the only external network request made by the Extension.

6. Permissions Explained

Permission Reason
bookmarks Read your Chrome bookmarks to display them in the Extension
storage Save pinned bookmarks, recent history, and theme preference locally
tabs Open bookmarks in a new tab and enable the Side Panel
sidePanel Display the persistent Side Panel in Chrome

7. Children's Privacy

This Extension does not knowingly collect any information from children under 13. It does not collect any personal information from any user of any age.
